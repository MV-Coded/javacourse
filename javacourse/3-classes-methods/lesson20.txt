static logic
logic (1) id= 1
logic (2) id= 1
ctor id= 100

Объяснение: При запуске программы сначала выполняется статический блок (выводит "static logic"). Затем создается экземпляр класса A с помощью конструктора new A(100). 
Перед выполнением тела конструктора последовательно выполняются все нестатические логические блоки в порядке их следования в коде. 
Первый логический блок выводит значение поля id, которое уже проинициализировано значением по умолчанию (1), так как инициализация поля private int id = 1; происходит перед выполнением логических блоков. 
Второй логический блок также выводит значение id, равное 1. После этого выполняется тело конструктора, где id присваивается значение 100, и выводится соответствующее сообщение.
